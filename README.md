# Zest

Zest is a recipe curator application where users can filter through and browse cuisine-specific dishes, and save their own creations to their account.

## Motivation
As someone who consistently bounces between health kicks and researching different ways to incorporate marshmellows into desserts, I realized that I've accumulated quite the laundry list of receipes saved across a multitude of different websites. It became cumbersome to trek through all the different sites, sifting through each archive to find the cuisine I was looking for. Wouldn't it be so much easier if I could save these by cuisine and search through them in a more predictable manner? Thus, Zest was born.

## Setup
To try out this project:

- Clone the GitHub repository locally to your computer
- In the command line, navigate to the root directory of the repository, and type the following: ```$ npm install```
- Then type the following: $ ```npm install —prefix client```
- The type: ```npm start —prefix client```
- Lastly, start the server: ```rails s```
